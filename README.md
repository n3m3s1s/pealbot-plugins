pealbot-plugins
===============

Plugin for the PealBot

admin.py: Gives the commands: deop, devoice, hdeop, hop, join, kill, op, operator, part, send, voice

greet.py: Plugin to greet new users that join the channel

config.py: Alternate configuration that enables the plugins and defines the Operator that will be able to execure the admin-commands

pealbot.py: Alternate main-bot that provides two more functions.
